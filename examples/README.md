# Examples for Java HTTP Client

This directory contains examples demonstrating how to use the Java HTTP Client API in Scala Native.

Codes in this directory are inspired by the following resources:

- <https://www.baeldung.com/java-9-http-client>
- <https://jenkov.com/tutorials/java-networking/httpclient.html>
- <https://openjdk.org/groups/net/httpclient/recipes.html>
